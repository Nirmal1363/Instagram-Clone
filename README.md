# Instagram Clone

A frontend design of an Instagram clone built with HTML and CSS, using JSON files for mock data.

## Features

- Responsive layout mimicking Instagram's interface
- Feed page with photo posts
- User profile page
- Navigation bar
- Mock data stored in JSON files

## Technologies Used

- HTML5
- CSS3
- JSON
